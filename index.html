Chrono Engine

Chrono Engine is a powerful, flexible, and intuitive tool for creating video games. It supports a wide range of game types and platforms. Chrono Engine uses ImGui for its user interface, providing a simple and easy-to-use interface for game developers. It also uses Direct3D 11 for rendering, with support for Direct3D 12 and OpenGL ES3, ensuring high performance and compatibility with a wide range of hardware.
